# kiritoaster-utility
A somewhat unpolished discord moderation bot thats inside of heavy beta. Currently inside of v1.0.0!
# How to use the bot
1: Invite the bot from this invite link. https://web.kiritoasty.us/kiritoaster-utility (important)
2: Give the bot a high role and make sure it has administrator so it can log moderation, and make prefix moderation commands.
3: View .help inside of the bot for more info.
# How to self host
1: Make a new folder on your self-hosted linux server called whatever you wanna make it named. Heres a small script that you can paste into your terminal if you need it.

```bash
# Create a folder, go inside it, init npm, install discord.js, and download script
mkdir whateveryouwanttonameit && cd whateveryouwanttonameit
npm init -y
npm install discord.js
curl -O https://web.kiritoasty.us/kiritoaster-utility/index.js
